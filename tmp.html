<h1 id="about-space-exploration-dashboard">About Space Exploration Dashboard</h1>

<p>This dashboard provides an interactive way to explore space-related data using NASA’s Open APIs. It was built using R Shiny and various other open-source tools.</p>

<h2 id="features">Features</h2>

<ul>

<li><strong>Astronomy Picture of the Day (APOD)</strong>: View NASA’s featured space image or video for any date, along with a detailed explanation.</li>

<li><strong>Near-Earth Objects (NEO) Tracker</strong>: Monitor and visualize asteroids that pass close to Earth.</li>

<li><strong>Mars Rover Photos</strong>: Browse the latest images from Mars rovers.</li>

<li><strong>ISS Tracker</strong>: Track the International Space Station’s current location in real-time.</li>

</ul>

<h2 id="data-sources">Data Sources</h2>

<p>All data is sourced from NASA’s Open APIs:</p>

<ul>

<li><a href="https://api.nasa.gov/">NASA Open APIs</a></li>

<li><a href="https://github.com/nasa/apod-api">APOD API</a></li>

<li><a href="https://neo.jpl.nasa.gov/">NEO Web Service</a></li>

</ul>
